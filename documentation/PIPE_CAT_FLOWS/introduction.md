# Pipecat Flows Introduction

Pipecat Flows is a framework for building structured, multi-step conversational experiences.

## What are Pipecat Flows?

It allows you to define complex conversational logic as a series of interconnected states and transitions. This is ideal for tasks like intake forms, troubleshooting guides, or guided sales flows.

## Key Features

* **State-Based Logic**: Easily manage complex conversation flows.
* **Declarative Syntax**: Define your flows using a simple, readable format.
* **Integrated Tools**: Seamlessly use tools and LLMs within your flows.
* **Context Management**: Automatically handle state and context across multiple turns.

## How it Works

You define a flow as a graph of states. Each state represents a specific point in the conversation. Transitions between states are triggered by user input, tool results, or other events.

## Getting Started

Check out the [Quickstart guide](/pipecat-flows/get-started/quickstart) to build your first flow.
