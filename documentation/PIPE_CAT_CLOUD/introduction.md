# Pipecat Cloud Introduction

Pipecat Cloud is the hosted platform for deploying, managing, and scaling your Pipecat agents.

## What is Pipecat Cloud?

It provides the infrastructure needed to run your Pipecat pipelines in a production environment without having to manage servers yourself.

## Key Features

* **Serverless Deployment**: Deploy your agents with a single command.
* **Auto-scaling**: Automatically scales your agents based on demand.
* **Monitoring & Logging**: Get insights into your agents' performance and debug issues.
* **Secure Secret Management**: Safely store your API keys and other sensitive data.
* **Global Infrastructure**: Run your agents close to your users for low latency.

## Why use Pipecat Cloud?

* **Focus on Development**: Spend more time building features and less time on DevOps.
* **Reliability**: Built on top of Daily's proven WebRTC infrastructure.
* **Scalability**: Handle thousands of concurrent calls with ease.

## Getting Started

To get started, sign up for an account at [pipecat.daily.co](https://pipecat.daily.co) and follow the [Quickstart guide](/pipecat-cloud/get-started/quickstart).
