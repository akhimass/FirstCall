---

Agentic applications demand near-instant agent availability to maintain user engagement and responsiveness. Pipecat Cloud manages the complexity of scaling these agent deployments in production environments.

## Core concepts
### Instances
An instance represents a single unit of compute that runs your agent.

### Instance pool
Making a deployment to Pipecat Cloud creates a managed pool of agent instances.
