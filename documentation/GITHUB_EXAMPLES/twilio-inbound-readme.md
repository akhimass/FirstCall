This project is a Pipecat-based chatbot that integrates with Twilio to handle inbound phone calls via WebSocket connections.

## How It Works
When someone calls your Twilio number:
1. Twilio sends WebSocket messages
2. Parse the WebSocket messages
3. Bot starts responding
